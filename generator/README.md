# CV Generator

## Requirements

* [Task](https://taskfile.dev/installation/)
* [Docker + compose](https://docs.docker.com/get-docker/)

## Description

Generates the CV as HTML and PDF from `data/cv.json` using Symfony and Twig.

## Install

```bash
task init
```

## Usage

List all available tasks:

```bash
task
```
